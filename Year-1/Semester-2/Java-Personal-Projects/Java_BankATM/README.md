# Simple ATM System (Java)

## Description

This is a simple Java console program that simulates basic ATM operations.
Users can deposit money, withdraw funds with a daily withdrawal limit, check their balance, and view transaction history.

The program also handles invalid input using exception handling.

---

## Features

* Deposit money
* Withdraw money
* Withdrawal limit per session
* Check account balance
* Transaction history
* Input validation using try-catch

---

## Program Menu

1. Deposit
2. Withdraw
3. Check Balance
4. Transaction History
5. Quit

---

## Concepts Used

* `while` loop for continuous program execution
* `switch` statement for menu selection
* `ArrayList` to store transaction history
* `try-catch` for error handling
* `Thread.sleep()` for countdown before exit
* `Scanner` for user input

---

## Example Transactions

* Deposited: $50
* Withdraw: $20
* Check Balance: $30
* Insufficient Balance Attempt
