# Simple SIM Card Loading System

A Java console program that simulates a SIM card loading system.

## Features
- Phone number validation
- SIM network detection
- Load amount validation
- Payment validation
- Error handling using loops

## Network Detection
- **GLOBE / TM** → Prefix range `0901 – 0950`
- **SMART / SUN** → Prefix range `0951 – 0999`
